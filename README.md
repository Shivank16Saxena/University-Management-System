# University-Management-System

Databases consisting of University Branches, Departments, Courses, Employees and Students.
Procedures created where Branch ID and Department ID will be given as input and information related to that branch and department will be displayed.
Created Report with Report Builder which will display information for a particular employee using Employee ID or for the whole Department using Department ID.
Created two forms using form builder which will take information from students and employees.
